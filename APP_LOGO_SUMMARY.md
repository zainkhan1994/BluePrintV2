## Logo Types and Sources

- Most logos were downloaded as PNG files from Clearbit's API
- Some logos were downloaded as SVG files from Simple Icons
- Two logos (Google Tasks and Go Kinetic) were created as placeholder SVGs

## Usage Notes

- All logos are saved in their respective folders as either `logo.png` or `logo.svg`
- PNG logos are bitmap images with transparency
- SVG logos are vector graphics that can be scaled to any size without loss of quality
- The logos are suitable for use as folder icons or within your blueprint visualizer

For any logo replacement needs, simply download the official logo from the app's website and replace the existing file, keeping the same filename.

## Additional Categories

If you need logos for additional categories of apps in the future, simply update the APPS list in the `download_app_logos.py` script and run it again.
